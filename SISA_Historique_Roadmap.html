Roadmap par phases (échéancier)

Phase 1 — Go‑live Intranet

  • Serveur SISA (R730XD), RAID1 SSD, UPS.
  • Site vitrine HTML (staging GitHub Pages).
  • Email Routing + boîtes IMAP (temporaire si besoin).
  • Workbench/Lanceur intégrés.

En cours

Phase 2 — SISA‑IA (v1)

  • vLLM/Ollama (serving), API interne.
  • RAG : Qdrant + FastAPI, citations & audit.
  • OpenWebUI, Nginx gateway, SSO/quotas.
  • Prometheus/Grafana (latence, tokens, charge).

Prochain

Phase 3 — Wi‑Fi & Services

  • Maillage Wi‑Fi (APs intérieurs + extérieur IP65).
  • VoIP FreePBX/Asterisk (TLS/SRTP, E911).
  • Ventoy (ISOs secours + scripts + SHA256).
  • PWA cache & sync opportuniste (faible connectivité).

Planifié

Phase 4 — Évolution

  • GPU plan (CPU → 1×GPU → 2×GPU).
  • Publication app (PWA/Capacitor) après stabilisation.
  • Marketing : Avantages SISA, SEO, carrousels.
